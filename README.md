# Slooze Data Science Take Home Challenge

## Project Overview
This project analyzes sales, purchases and inventory data to optimize stock management and extract business insights.

## Objectives
- Demand forecasting
- ABC inventory classification
- Reorder point calculation
- Lead time analysis
- Inventory optimization

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## How To Run

1. Install Anaconda
2. Open Jupyter Notebook
3. Place all CSV files in project folder
4. Open inventory_analysis.ipynb
5. Run all cells

## Results Summary
- Average Daily Demand: ~40,000 units
- Reorder Point: ~311,000 units
- Inventory Turnover Ratio: 0.53
- Identified overstock risk and optimization opportunities
